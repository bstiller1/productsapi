# Inventory API  
## Instructions  
`javac ProductsapiApplication.java`  
`java ProductsapiApplication`  

Use Postman to load the *URL* and *Routes* below:  
`http://localhost:8080`  


## Routes  
**GET** `/api/products` - Show all products  
**POST** `/api/products` - Add a product  
**PUT** `/api/products/1` - Update a product  
**DELETE** `/api/product/1` - Remove a product
